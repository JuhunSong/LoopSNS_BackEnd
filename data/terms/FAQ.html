자주 묻는 질문



회원


● 어떻게 가입할 수 있나요?


회원가입 방법은 다음 과정을 따라주시면 됩니다.

⇒

1. Loop 어플리케이션을 Google Playstore에서 다운로드 받아주세요.

2. 구글 혹은 카카오 계정으로 로그인 하세요.

3. 구글 혹은 카카오 계정 인증이 완료되면 로그인이 완료됩니다.



● 어떻게 로그아웃을 진행하나요?


로그아웃 방법은 다음 과정을 따라주시면 됩니다.

⇒

1. 'Home' 화면에서 좌측에 있는 'Profile'로 이동하세요.

2. 오른쪽 상단의 톱니바퀴 모양 설정창을 클릭하세요.

3. '계정관리' 부분에서 '로그아웃' 버튼을 클릭하면 로그아웃이 가능합니다.



● 어떻게 다른 아이디로 로그인을 진행하나요?


다른 아이디로 로그인 방법은 다음 과정을 따라주시면 됩니다.

⇒

1. 'Home' 화면에서 좌측에 있는 'Profile'로 이동하세요.

2. 오른쪽 상단의 톱니바퀴 모양 설정창을 클릭하세요.

3. '계정관리' 부분에서 '로그아웃' 버튼을 클릭하세요.

4. 로그아웃 후 시작화면에서 새로운 계정을 선택하여 로그인하세요.



● 어떻게 탈퇴하나요?


회원탈퇴 방법은 다음 과정을 따라주시면 됩니다.

⇒

1. 'Home' 화면에서 좌측에 있는 'Profile'로 이동하세요.

2. 오른쪽 상단의 톱니바퀴 모양 설정창을 클릭하세요.

3. '계정관리' 부분에서 '계정 삭제' 버튼을 클릭하세요.

4. 마지막으로 계정 삭제를 위해 '삭제'를 입력 후 '삭제' 버튼을 클릭하세요.



● 미성년자도 가입할 수 있나요?


구글 혹은 카카오 계정을 소유하고 있다면 계정 생성이 가능합니다.



● 어떻게 영어 혹은 한국어로 언어 설정이 가능한가요?


언어 설정 방법은 다음 과정을 따라주시면 됩니다.

⇒

1. 'Home' 화면에서 좌측에 있는 'Profile'로 이동하세요.

2. 오른쪽 상단의 톱니바퀴 모양 설정창을 클릭하세요.

3. '언어 설정' 버튼을 클릭하고 원하는 언어를 선택하세요.

4. 재시작을 진행한다는 알림창에 '예'를 클릭하여 언어 설정을 완료하세요.



● 어떻게 프로필 사진과 이름을 변경할 수 있나요?


프로필 수정 방법은 다음 과정을 따라주시면 됩니다.

⇒

1. 'Home' 화면에서 좌측에 있는 'Profile'로 이동하세요.

2. 오른쪽 상단의 톱니바퀴 모양 설정창을 클릭하세요.

3. '개인 정보 설정' 부분에서 '개인정보 수정' 버튼을 클릭하세요.

4. 이름과 프로필 사진을 수정한 후 저장 버튼을 클릭하세요.



● 작성한 게시글을 삭제하고 싶습니다.


작성한 게시글 삭제 방법은 다음 과정을 따라주시면 됩니다.

⇒

1. 삭제하고자 하는 게시글을 클릭해 '게시글 상세보기' 화면으로 이동하세요.

2. 오른쪽 상단에 더보기 버튼을 클릭하세요.

3. 나온 메뉴 중 '게시글 삭제' 버튼을 클릭하여 삭제하세요.

4. 게시글 삭제는 본인의 게시글만 가능합니다.



● 작성된 게시글을 신고하고 싶습니다.


게시글 신고 방법은 다음 과정을 따라주시면 됩니다.

⇒

1. 신고하고자 하는 게시글을 클릭해 '게시글 상세보기' 화면으로 이동하세요.

2. 오른쪽 상단에 더보기 버튼을 클릭하세요.

3. 나온 메뉴 중 가장 아래에 있는 '신고' 버튼을 클릭하여 신고하세요.

4. 신고된 게시글은 검토 후 제재 또는 삭제가 진행됩니다.
